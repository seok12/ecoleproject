서비스 소개

개발 배경

'수화'라는것 자체는 우리에게 무척 익숙하지만, 실상 '수화'에 대해 잘 아는 이는 드물다. 그래서 수화를 배우고 싶은 사람들에게 더 나은 교육환경을 제공하고, 후천적으로 장애를 얻은 사람들이 수화를 배워야할 때 교육이 가능하다.

즉, 수화에 대한 접근성이 높아질 것을 예상으로 개발하게 되었습니다.

구동 과정

  • 1. 사용자의 디바이스를 통해 자신이 익힌 수화 동작을 구현한다.
  • 2. 스마트폰이 Microsoft Azure 서비스와 연동되어 기존에 있던 수화 단어와 비교 분석을 한다.
  • 3. 정상적인 수화 동작이라면 해당 단어를 표기하는 AI 학습 모델이다.

    • Microsoft Azure 서비스 중 Object Detection과 Classification에 수화 데이터를 입력하는 과정을 보여주고있다.

    기대 효과

    서비스를 반복해서 이용한다면, 수화를 더욱 편하고 빠르게 학습 할 수 있을 것입니다.

    그리고 수화를 익히고 있는 사람은 매우 적기 때문에, AI 수화 학습 서비스가 그것을 대체할 수 있다.
